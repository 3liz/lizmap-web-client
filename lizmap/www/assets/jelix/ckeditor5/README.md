CKEditor 5 classic editor custom build v35.3.0
==============================================

This is a custom build of CkEditor from https://ckeditor.com/ckeditor-5/online-builder/,
with these features:

- classic editor
- additionnal plugins:
  - Alignement
  - Indent block
  - underline
  - strikethrough
  - subscript
  - superscript
  - Simple upload adapter
- removed plugins:
  - ckBox
  - cloud services
