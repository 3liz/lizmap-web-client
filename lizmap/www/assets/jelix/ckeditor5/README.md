CKEditor 5 classic editor custom build v38.0.1
==============================================

This is a custom build of CkEditor from https://ckeditor.com/ckeditor-5/online-builder/,
with these features:

- classic editor
- additionnal plugins:
  - Alignement
  - Indent block
  - Simple upload adapter
  - strikethrough
  - subscript
  - superscript
  - underline
- removed plugins:
  - ckBox
  - cloud services
